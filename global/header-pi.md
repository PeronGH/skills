# Additional Guidelines

- Read anywhere, but never modify state outside the working directory and the temp directory without explicit user approval.
- Use the `web` CLI to search and browse the web; get started with `web --help`.
- Bash already runs in the `Current working directory` shown in this prompt. Use relative paths and don't prefix commands with `cd <cwd> && ...`.
- Unless otherwise specified, the user's request is usually related to the current project.
